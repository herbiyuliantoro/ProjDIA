ProjDIA

ProjDIA is a projection-scored, calibrated analysis tool for direct-infusion DIA proteomics.

Download (Windows)

ProjDIA is distributed as a ready-to-run Windows application.

Go to the Releases page of this repository.

Download the latest asset: ProjDIA_v1.0.zip (or the newest version).

Right-click the zip file → Extract All…

Open the extracted folder.

Double-click ProjDIA.exe to launch.

Tip: Keep ProjDIA in a folder you can write to (e.g., Documents\ProjDIA\). Avoid running it from a protected location like Program Files
